# Git  

Git có 3 state là: Working directory, Staging Area, Repository
Các comment khi làm việc với git:

+ Khởi tạo repo local: git init
+ Tạo repo GitHub và liên kết tới reppo local: **git remote add origin <url>**
+ Thêm file vào staging: **git add .** hoặc **git add file name** 
+ Commit file (đẩy file lên repo): **git commit -m"message"**
+ Push code lên GitHub: **git push origin main**

# JavaScript
- Comment chạy file JS: **node tên file** hoặc **node <đường dẫn tới file>**
- Các từ khóa khai báo biến: var, let, const
  + var cho phép khai báo lại 
  + let ko cho phép khai báo lại
  + const hằng số, chỉ dùng khi ko có nhu cầu thay đổi 
- Data type: string, number, boolean, undefined, Null, Object
- Câu điều kiện:
  + if
  + if...else
  + if..else if..else
  + switch...case
- Vòng lặp:
  + for (i)
  + for (of)
  + for (each)
  + for (in)
  + while
  + do..while
